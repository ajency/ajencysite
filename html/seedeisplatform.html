SeedEISPlatform.com
A crowdfunding platform in London.

An invite only crowdfunding platform that has already raised close to a million pounds for ten different businesses. The site is a closed community that accepts only qualified investors and those businesses which have a valid hmrc status as approved by the UK government. It combines features of a typical crowdfunding platform with highly specialized and compliance ready tools for wealth managers, angel funds and incubators to participate in the UK government's EIS and SEIS schemes.

While responsive websites are quite the norm today, building a responsive web application comes with its own set of challenges. Key challenges we had to address :
  • 01

    Long form

    Having users fill long forms is always a challenge. More so, when they are using a mobile phone.

  • 02

    User Dashboard

    How do you decide which modules show up on a mobile dashboard and which don't?

  • 03

    Mobile message board

    Message board and file uploads working across device types. Especially with each platform and device maker having different permissions for file upload in a browser.

 

 

User Management Console

We are proud of the user management console we have built. It allows the site manager to define different user types on the fly, and then assign them access to different parts of the site. Everything from the site menu to the content, and the actions visible for a user, can be controlled. This gives great flexibility to grow and extend the system with time.
